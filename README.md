# aviationweather

Node.js engine for global aviation weather. Written with typescript.

## API Coverage

### AVIATION WEATHER CENTER

- METARs
- Aircraft Reports
- TAFs
- AIR/SIGMETs
- G-AIRMETs
- STATION INFO

### AVT7

- METAR
